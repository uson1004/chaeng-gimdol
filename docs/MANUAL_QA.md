# 챙김돌 Phase 1 Manual QA

## 실행 환경

| 환경 | 상태 | 메모 |
|---|---|---|
| API 26 emulator | 미실행 | 현재 로컬 SDK에 시스템 이미지 없음 |
| API 33 emulator | 미실행 | 현재 로컬 SDK에 시스템 이미지 없음 |
| API 36 emulator/device | 미실행 | 플랫폼만 설치되어 있고 시스템 이미지 없음 |
| API 37 Pixel 10 Pro AVD | 통과 | 자동 UI 테스트 11건, 온보딩·홈 시각 확인 |

API 26·33·36 행은 해당 시스템 이미지를 준비한 뒤 출시 전에 실행해야 합니다.

## 핵심 흐름

- [ ] 새 설치에서 온보딩이 열린다.
- [ ] 기본 물품 세 개를 선택하면 활성 물품 세 개가 생성된다.
- [ ] 홈에서 한 번의 탭으로 수동 세션을 시작한다.
- [ ] 물품을 `미확인`, `챙김`, `해당 없음`으로 구분할 수 있다.
- [ ] 미확인 물품을 남기고 완료하면 확인 대화상자가 열린다.
- [ ] 완료하면 위치 포그라운드 서비스와 알림이 제거된다.
- [ ] 앱을 종료하고 다시 열어도 진행 중 세션에 `챙김 계속하기`로 복귀한다.

## 권한

- [ ] 알림 권한을 거부해도 수동 체크가 가능하다.
- [ ] 위치 권한을 거부하면 `수동으로 시작`이 보인다.
- [ ] 대략적 위치만 허용해도 앱이 종료되지 않는다.
- [ ] 설정에서 출발 감지를 끄면 실행 중 서비스가 중지된다.
- [ ] 시스템 설정에서 권한을 다시 허용하면 다음 세션부터 감지가 가능하다.

## 위치와 알림

- [ ] 위치 감지 중 포그라운드 알림이 보인다.
- [ ] 세션 시작 10분 전에는 이동 알림이 오지 않는다.
- [ ] 시작점에서 100m 미만 이동에는 알림이 오지 않는다.
- [ ] 정확도 50m 초과 위치는 이탈로 판단하지 않는다.
- [ ] 한 세션에서 이동 알림은 최대 한 번만 온다.
- [ ] 알림을 누르면 진행 중 세션으로 이동한다.
- [ ] 이미 끝난 세션 알림을 열면 홈에서 종료 안내를 보여준다.

## 접근성

- [ ] TalkBack이 물품 이름과 상태를 한 행으로 읽는다.
- [ ] 장식용 챙김돌 표정을 반복해서 읽지 않는다.
- [ ] 글꼴 크기 200%에서 물품명과 `다 챙겼어요`가 보인다.
- [ ] 주요 버튼의 터치 영역이 최소 48dp다.

## 데이터와 개인정보

- [ ] 앱 데이터 삭제는 확인 대화상자를 거친다.
- [ ] 삭제 후 물품·세션·설정이 초기화되고 온보딩으로 이동한다.
- [ ] Room 테이블과 로그에 위도·경도가 없다.
- [ ] 앱은 계정이나 외부 서버 없이 동작한다.
