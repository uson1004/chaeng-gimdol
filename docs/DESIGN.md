# 챙김돌 Signal Buddy 디자인 계약

## 콘셉트

Signal Buddy는 `어반 시그널 65% + 다정한 수호돌 35%` 조합입니다. 이동 중 즉시 읽히는 높은 대비와 짧은 정보 구조를 유지하면서, 코랄색 조약돌 캐릭터와 비난하지 않는 문구로 알림의 거부감을 낮춥니다.

## 색상

| 토큰 | 값 | 용도 |
|---|---:|---|
| Charcoal | `#171914` | 본문과 아이콘 |
| Signal Lime | `#D7FF3F` | 화면의 단일 주요 행동과 확인 상태 |
| Buddy Coral | `#FF7478` | 캐릭터 |
| Deep Lilac | `#775093` | 선택·보조 상태 |
| Warm Ivory | `#F1F0E8` | 기본 배경 |

강조색은 한 화면에 최대 두 개만 사용합니다. 상태는 색상만으로 전달하지 않고 텍스트와 접근성 상태 설명을 함께 제공합니다.

## 타이포그래피와 간격

- Display: 30sp, 800~900
- Title: 24sp, 800~900
- Section: 18sp, 800
- Body: 16sp, 500~700
- Label: 13sp, 700~800
- Caption: 12sp, 500~700
- 화면 좌우 여백: 20dp
- 주요 버튼: 최소 52dp
- 터치 영역: 최소 48dp

별도 폰트 파일은 추가하지 않고 Android 시스템 한글 폰트를 사용합니다.

## 캐릭터 크기

| 화면 | 크기 |
|---|---:|
| 온보딩 | 112~144dp |
| 오늘 홈 | 96~120dp |
| 챙김 세션 | 64~80dp |
| 완료 | 104~128dp |
| 내 물건·설정 | 숨김 |

캐릭터가 장식일 때는 TalkBack 트리에서 제거합니다.

## 정보 구조

Phase 1 하단 내비게이션은 다음 세 항목으로 고정합니다.

1. 오늘
2. 내 물건
3. 설정

분실 검색과 외부 공공데이터는 검증되지 않은 빈 기능으로 노출하지 않고 Phase 2로 둡니다.

## 접근성

- Charcoal과 Warm Ivory, Charcoal과 Signal Lime 조합으로 높은 대비를 유지합니다.
- 물품 행은 이름과 `미확인`, `챙김`, `해당 없음` 상태를 하나의 의미 단위로 읽습니다.
- 동적 글꼴 200%에서 물품명과 주요 CTA가 보여야 합니다.
- 오류는 입력을 보존하고 `다시 시도` 같은 실행 가능한 복구 행동을 제공합니다.
- 감지 결과를 확정 사실로 표현하지 않고 `이동한 것으로 보여요`라고 안내합니다.
